# 线性纵向建模笔记

---

## 一、纵向研究的方法论诉求

### 1.1 独立性假设的破裂

经典 OLS 的核心前提是观测值**独立同分布（i.i.d.）**，残差协方差矩阵须满足：

$$\Omega = \sigma^2 I$$

纵向数据从设计层面就违背了这一假设：同一受试者在不同时间点的得分受相同的基因、人格、环境及历史因素影响，必然存在系统性相关性。

**强行使用 OLS 的后果：**

- **正向自相关** → 系统性低估标准误 → $t$/$F$ 统计量虚高 → 大量假阳性（I 类错误）
- 个体间系统性变异被误判为随机噪声，掩盖真实个体差异

### 1.2 变化的测量方式

- **原始变化得分 (Raw Change Score)**：$D = Y_2 - Y_1$；简单直接，但受基线水平影响，测量可靠性可能低于原始值
- **余数化变化得分 (Residualized Change Score)**：将后期得分对基线回归，提取无法由起始状态解释的部分；本质是在建模"在特定背景下预期的差异"

> 这一逻辑差异在自回归交叉滞后模型（ARCL）与潜曲线模型（LGCM）的对立中得到深化。

## 二、矩阵水平的解决方案：协方差结构参数化

### 2.1 线性混合模型（LMM）的矩阵解构

对第 $i$ 个个体，观测向量 $\mathbf{y}_i$ 的模型结构为：

$$\mathbf{y}_i = X_i\beta + Z_i\mathbf{b}_i + \boldsymbol{\epsilon}_i$$

- **$X_i\beta$**：群体水平的固定效应（Fixed Effects）
- **$Z_i\mathbf{b}_i$**：个体偏离群体的独特轨迹（随机效应，Random Effects）

模型隐含的协方差矩阵 $V_i$：

$$V_i = \text{Cov}(\mathbf{y}_i) = Z_i G Z_i' + R_i$$

- **$G$**：随机效应的协方差阵，代表个体间在截距和斜率上的异质性
- **$R_i$**：个体内残差阵，代表特定时间点的测量误差或短期波动

### 2.2 纵向 SEM（LSEM）的隐含矩阵

SEM 框架下，各时间点被定义为潜在增长因子的指标，核心测量方程为 $\mathbf{y} = \Lambda\eta + \boldsymbol{\epsilon}$，观测变量的协方差阵为：

$$\Sigma(\theta) = \Lambda \Psi \Lambda' + \Theta$$

- **$\Lambda$（载荷矩阵）**：固定时间的尺度，定义变化的形态（线性、二次项等）
- **$\Psi$（潜在因子协方差阵）**：数值上等价于 LMM 中的 $G$ 矩阵，捕捉潜在截距和斜率的方差及其相关性
- **$\Theta$（残差阵）**：等价于 LMM 中的 $R$ 矩阵，捕捉测量误差

通过将变量间的相关性转移到潜在因子 $\eta$ 空间，SEM 在数学上"纯化"了增长因子的估计，消除了独立性违背带来的估计偏倚。

## 三、模型家族对比：GEE vs. LMM vs. LSEM

### 3.1 广义估计方程（GEE）— 群体平均的边际视角

- **解释层次**：估计协变量每变化一个单位，**整个人群平均响应**的变化程度
- **处理相关性**：指定"工作相关矩阵"$R(\alpha)$，利用**夹心方差估计（Sandwich Estimator）**提供稳健标准误修正
- **优势**：即使工作相关矩阵设定错误，只要均值方程正确，参数估计依然**一致**
- **适用场景**：公共卫生和政策评估，关注人群平均趋势

### 3.2 线性混合模型（LMM）— 个体特异的条件视角

- **解释层次**：基于特定个体随机效应值的**条件期望**
- **逻辑差异**：对于非线性模型（如逻辑回归），LMM 的固定效应估计值通常大于 GEE 的边际估计值——LMM 是控制了个体水平潜在变异后的纯效应，GEE 被个体间异质性稀释
- **适用场景**：进行个体化预测或理解个体差异来源

### 3.3 纵向 SEM / 潜曲线模型（LGCM）— 潜在过程的视角

- **处理测量误差**：SEM 的杀手锏——LMM 假设观测值即为真实值，而 SEM 允许使用多个指标定义潜在构造，在估计增长率时自动剔除测量不可靠性
- **灵活性**：可轻松整合中介路径、调节路径以及多过程并行增长（Parallel Process）

### 3.4 四维对比速查

- **解释层次**：GEE = 群体平均趋势 / LMM = 个体条件轨迹 / LSEM = 潜在过程轨迹
- **数据格式**：GEE/LMM = 长格式（Long） / LSEM = 宽格式（Wide）
- **统计基础**：GEE = 拟似然 / LMM = 极大似然（ML/REML） / LSEM = 协方差结构分析
- **缺失值**：GEE 依赖 MCAR / LMM 允许 MAR / LSEM 允许 MAR（FIML）
- **测量误差**：GEE 无法显式建模 / LMM 视为残差一部分 / LSEM 通过多指标纯化

## 四、局部结构方程模型（Local SEM / LSEM）

### 4.1 核心逻辑

传统加入乘法交互项的做法预设调节效应是**线性的**。LSEM 沿着连续调节变量（Moderator, $A$）的轴线，在每一个"焦点"（Focal point, $a$）上拟合一个局部模型，类似于非参数回归中的**核平滑（Kernel Smoothing）**。

核心权重公式：

$$w_i(a) = \frac{1}{\sqrt{2\pi}\, h} \exp\!\left(-\frac{(A_i - a)^2}{2h^2}\right)$$

- **$h$（带宽 Bandwidth）**：距离焦点 $a$ 越近的观测单位，在该点局部估计中占有越大的权重

### 4.2 LSEM 的诊断功能

- **参数随焦点变化**：潜在增长因子的均值 $\alpha(a)$、载荷 $\lambda(a)$ 乃至残差 $\theta(a)$ 都可以呈现非线性波动
- **检测测量等价性细微破裂**：若某个指标的载荷随年龄显著增加，则该点处构型等价性（Configural Invariance）可能受损

## 五、潜曲线模型（LGCM）深度解析

### 5.1 均值结构与协方差结构的联合估计

- **均值结构（$\alpha$）**：反映人群的平均增长趋势
- **协方差结构（$\Psi$）**：反映个体间差异
- **约束与自由度**：LGCM 通常是不饱和模型，对均值形态（如必须符合直线）提出假设。若模型拟合优度（CFI、TLI、RMSEA）不佳，说明简单函数形态无法解释观测到的均值偏移

### 5.2 时间的编码与潜基底模型

- **中心化（Centering）的影响**：
  - 第一个时间点载荷设为 0 → 截距因子代表**基线水平**
  - 中间点设为 0 → 截距代表**发展中期水平**
- **潜基底模型（Latent Basis Change Model, LBCM）**：固定第一个载荷为 0、最后一个载荷为 1，中间点载荷作为**自由参数由数据估算**，反映个体在特定时间段内完成总改变量的百分比

### 5.3 测量等价性的纵向检验

进行增长建模前，须通过多组验证性因子分析（CFA）证明跨时点测量等价性：

- **弱等价性（Metric Invariance）**：确保不同时点潜在因子的单位一致（载荷相等）
- **强等价性（Scalar Invariance）**：比较均值的**必要前提**（截距相等）

> 若强等价性违背，不宜直接运行 LGCM——此时观察到的增长可能是指标本身"漂移"的结果，而非潜构造的真实变化。

## 六、进阶纵向动力学：滞后效应与变化分数

### 6.1 自回归交叉滞后模型（ARCL）

回答核心问题：$X$ 对 $Y$ 的跨时预测力是否超过 $Y$ 对 $X$ 的预测力？

- **自回归路径**：捕捉特征的**稳定性**
- **交叉滞后路径**：捕捉变量间的**动态驱动关系**
- **致命缺陷**：传统 ARCL 混淆了受试者内和受试者间的变异。由人格（个体间差异）导致的虚假相关可能被误读为动态因果

> **解决方案**：使用**带有随机截距的交叉滞后模型（RI-CLPM）**，将个体间稳定差异从交叉滞后路径中剥离。

### 6.2 潜在变化分数模型（LCSM）

将纵向过程解构为：**当前状态 = 前一状态 + 瞬时改变量**

- **矩阵实现**：通过引入"幻影变量"（Phantom Variables）分离出每一时段的变化量 $\Delta Y$
- **比例变化（Proportional Change）**：当前变化量受前一状态影响（反馈机制），可模拟增长的加速或减缓
- **双重变化模型**：同时包含恒定增长分量和比例增长分量，适合捕捉生物发育或复杂社会现象的演进

## 七、复杂数据情形下的矩阵稳健性

### 7.1 缺失数据处理：FIML 与 MI

- **全信息极大似然法（FIML）**：直接计算每一个体基于其**可用数据点**的似然函数，不删除观测。只要缺失机制满足 MAR（随机缺失），参数估计一致且高效。适用于 SEM 框架
- **多重插补（MI）**：对于 GEE 这类非似然估计法，MI 是处理 MAR 的主要手段

### 7.2 非正态响应变量的矩阵修正

当输出变量是二分类、有序分类或计数数据时，传统连续变量 LGCM 会失效：

- **阈值模型（Threshold Model）**：引入潜在连续变量 $y^*$ 与观测分类变量 $y$ 的映射，建立基于 Probit 或 Logit 连接函数的测量方程
- **加权最小二乘法（WLS/DWLS）**：处理大样本分类数据时比 ML 具有更高计算效率，权重矩阵 $W$ 基于多斜相关（Polychoric Correlations）的渐近协方差阵

## 八、建模决策框架

- **关注整个人群的平均趋势？** → GEE
- **关注个体演进的独特性？** → LMM / LSEM
- **观测指标存在显著测量误差？** → SEM（含测量模型）
- **调节效应可能是非线性的？** → 局部 SEM（LSEM）
- **建模水平的稳定演进轨迹？** → LGCM
- **建模基于前一状态的动态反馈？** → LCSM
- **数据存在 MAR 缺失？** → FIML（SEM）或 MI（GEE）
- **结局为分类/计数变量？** → 阈值模型 / DWLS

> **总结**：通过将宏观的独立性假设违背转化为微观的协方差矩阵参数化，现代纵向模型提供了一套极其精密且强大的工具箱。未来方向包括动态结构方程模型（DSEM）对密集型数据的处理，以及对零膨胀计数等非典型分布的深度兼容。

---

## 九、假设失效的后果全景

- **独立性 (Independence)**：残差项之间存在系统相关性 → 参数估计不具有效性，统计推断失效
- **标准误 (Standard Error)**：模型倾向于低估估值的变异 → 置信区间过窄，$p$ 值虚假降低，假阳性激增
- **异质性 (Heterogeneity)**：未建模的个体差异被计入误差 → 无法区分"组内变化"与"组间差异"
- **缺失数据 (Missing Data)**：OLS 无法处理非平衡的数据结构 → 导致样本偏差和信息损失

## 十、变化的经典测量方案

### 10.1 增量分数（Difference / Gain Scores）

直接用后测减去前测，作为因变量对预测变量 $X$ 回归：

$$Y_{i,2} - Y_{i,1} = \beta_0 + \beta_1 X_i + e_i$$

- **目标**：捕捉个体指标变化的**绝对量**（"谁增加得最多？"）
- **优势**：在随机对照试验（RCT）中直接反映个体水平的演变
- **局限**：可靠性悖论——当组件相关性高时，增量的测量可靠性反而低于原始指标

### 10.2 残差化增量（Residualized Change / ANCOVA）

以前测作为协变量预测后测，提取无法由起始状态解释的剩余变异：

$$Y_{i,2} = \beta_0 + \beta_1 Y_{i,1} + \beta_2 X_i + e_i$$

- **目标**：在给定初始状态的情况下，衡量**相对于预期的偏离**（"谁的表现超出了预期？"）
- **局限**：易受**向平均数回归（Regression to the Mean, RTM）**的干扰，在非随机研究中易产生偏误

### 10.3 罗德悖论（Lord's Paradox）

> 罗德（1967）发现：在同一组数据上，增量分数法与 ANCOVA 可能得出**完全相反的结论**。

经典案例：大学食堂膳食对男女生体重的影响，基线与随访时男女体重分布完全一致：

- **增量分数法**：男女平均变化均为 0 → 结论：无性别差异
- **ANCOVA**：控制基线后，相同初始体重的男生预期后测体重大于女生 → 结论：膳食对男生有显著正向影响

**本质**：增量分数法强制基线回归系数为 1，ANCOVA 则估计该系数（通常 < 1）并对初始差异进行"校正"。在非随机分组的观察研究中，选择方法取决于研究者对因果机制的假设。

### 10.4 方法选择实操指南

- **随机对照试验（RCT）**：两种方法通常结果一致。随机分配保证各组基线在统计上相等，向均值回归效应在组间相互抵消；ANCOVA 在检验力上往往更优
- **观察性研究（组别非随机，如性别、社会阶层）**：必须谨慎！各组本来就不在同一起跑线，此时：
  - 想看**总效应（绝对变化量）** → 增量分数
  - 想看**直接效应（控制起点后的相对变化）** → ANCOVA，但需确保"控制基线"符合因果假设，否则产生 RTM 偏误
- **核心提醒**：change score 擅长回答"变了多少"，不擅长回答"变化过程长什么样"或"不同人的变化速率是否不同"；对于后者应转向 MLM 或 LGM

## 十一、多层线性模型（MLM）深度解析

### 11.0 核心框架：时间是 Level 1，个体是 Level 2

MLM 将纵向数据写成两层结构：**Level 1** 是个体内随时间变化的观测，**Level 2** 是个体之间的差异。基础线性增长模型的 Level 1 方程：

$$y_{it} = \pi_{0i} + \pi_{1i}\,\text{Time}_{it} + \varepsilon_{it}$$

- **$\pi_{0i}$**：个体 $i$ 的起点（截距）
- **$\pi_{1i}$**：个体 $i$ 的变化速率（斜率）

再将这两个参数写成"总体均值 + 个体偏差"，即得到完整的多层/随机效应表达。

### 11.1 随机效应方差分析模型（Random-effects ANOVA / Null Model）

最简无条件模型——不含任何预测变量，仅将观测值分解为三层：

$$y_{ij} = \gamma_{00} + u_{0j} + r_{ij}$$

- **$\gamma_{00}$（固定效应）**：总体均值（Grand Mean），对所有个体和群体均相同
- **$u_{0j}$（群体随机效应）**：第 $j$ 个群体的平均水平与总体均值的偏差；同组内所有成员共享同一个值
- **$r_{ij}$（个体残差）**：个体相对于其所属群体平均水平的偏差

**统计假设：**

- $u_{0j} \sim N(0, \tau_{00})$，$r_{ij} \sim N(0, \sigma^2)$，两者相互独立（跨层级不相关）

**三条数学推论：**

- **期望值**：$E(y_{ij}) = \gamma_{00}$（随机部分期望为 0，观测均值即固定截距）
- **总方差**：$\text{Var}(y_{ij}) = \tau_{00} + \sigma^2$（总变异 = 组间方差 + 组内方差，这正是 MLM 的**方差分量分解**）
- **组内协方差**：$\text{Cov}(y_{ij},\, y_{i'j}) = \tau_{00} \quad (i \neq i')$

  同组内不同个体共享同一个 $u_{0j}$，因此得分相关，相关性大小正是组间方差 $\tau_{00}$。只要 $\tau_{00} > 0$，同组个体就不再独立 → 这正是普通 OLS 处理嵌套数据会产生偏差的根本原因。

**组内相关系数（ICC）**：

$$\text{ICC} = \frac{\tau_{00}}{\tau_{00} + \sigma^2}$$

ICC 衡量"总变异中有多大比例来自群体间差异"，同时量化了数据依赖性的强度。ICC 较高 → 独立性受损严重 → 必须使用 MLM。

**为什么叫"随机效应 ANOVA"？**

传统 ANOVA 将组别视为**固定因子**（关心特定组的差异）；随机效应 ANOVA 将组别视为从总体中**随机抽取的样本**（关心组间波动的分布），目的是分解方差而非比较特定组均值。空模型（Null Model）没有任何自变量，是 MLM 最简形式，专门用于估计 ICC 并判断是否有必要进行多层建模。

**协方差矩阵 $V_j$ 的显式结构（以 3 人小组为例）：**

$$\mathbf{V}_j = \begin{bmatrix} \tau_{00} + \sigma^2 & \tau_{00} & \tau_{00} \\ \tau_{00} & \tau_{00} + \sigma^2 & \tau_{00} \\ \tau_{00} & \tau_{00} & \tau_{00} + \sigma^2 \end{bmatrix}$$

- **对角线元素** $\tau_{00} + \sigma^2$：观测值的总方差
- **非对角线元素** $\tau_{00}$：组内任意两个个体之间的协方差

只要 $\tau_{00} > 0$，非对角线元素就不为 0，组内观测值就不是独立的。这是 OLS 处理嵌套数据时为何会产生偏差的矩阵级解释——OLS 隐含地假设 $\mathbf{V}_j = \sigma^2 I$（即非对角线全为 0）。

**ICC 的局限性（"不靠谱"的情形）：**

- **二分结果变量**：Logistic 等模型中不存在独立的 Level-1 方差 $\sigma^2$，且方差具有异质性，没有统一接受的 ICC 计算标准
- **含随机斜率时**：组内相关性不再是常数，而是随自变量 $x$ 的取值而变化，单一数字的 ICC 总结失效
- **小群体样本量**：每组观测数很少时，估计出的群体均值含有较大测量误差，组间效应估计产生偏差

### 11.2 随机截距模型（Random Intercept Model）

在 null model 基础上引入时间预测变量，Level 1 + Level 2 联合表达：

$$Y_{ij} = \beta_{0j} + e_{ij}, \qquad \beta_{0j} = \gamma_{00} + u_{0j}$$

还原形式：$Y_{ij} = \gamma_{00} + u_{0j} + e_{ij}$，与 null model 结构相同，区别在于此处 $e_{ij}$ 中已被 Level 1 预测变量解释了部分变异。

### 11.2 随机斜率模型（Random Slope Model）

个体不仅起点不同，随时间变化的速率（斜率）也各不相同：

$$\text{Level 1:} \quad Y_{ti} = \beta_{0i} + \beta_{1i}(\text{Time}_{ti}) + r_{ti}$$

$$\text{Level 2:} \quad \beta_{0i} = \gamma_{00} + u_{0i}, \qquad \beta_{1i} = \gamma_{10} + u_{1i}$$

### 11.3 MLM 参数解读

- **$\gamma_{00}$（平均截距）**：样本在时间原点（如基线）的平均状态
- **$\gamma_{10}$（平均斜率）**：样本随时间变化的平均速率（增长趋势）
- **$\sigma^2_{u0}$（截距方差）**：个体在初始状态上的异质性程度
- **$\sigma^2_{u1}$（斜率方差）**：个体在变化轨迹上的异质性程度
- **$\sigma_{u01}$（截距-斜率协方差）**：初始状态与变化速度之间的关联（如"高原效应"：起点高的人增长反而更慢）

### 11.4 R/lmer 代码示例（GPA 场景）

```r
# 随机截距模型：每个学生起点不同，斜率固定
m1 <- lmer(gpa ~ time + (1 | student), data = longdat)

# 随机截距 + 随机斜率 + 性别交互：起点和增长速率均因人而异
m2 <- lmer(gpa ~ time * sex + (time | student), data = longdat)
```

- `(1 | student)`：只允许截距随个体变化
- `(time | student)`：截距和时间斜率均随个体变化（即随机截距 + 随机斜率）
- 在 LGM 语言中，"预测截距和斜率"对应 HLM 里的**主效应 + time 交互项 + 随机截距/斜率**

### 11.5 读结果时的四个关注点

- **Fixed effect of time**：平均增长/下降趋势
- **Var(random intercept)**：个体起点差异有多大
- **Var(random slope)**：个体增长速率差异有多大
- **Cov(intercept, slope)**：起点高的人后来变快还是变慢（正 = 马太效应，负 = 高原效应）

### 11.6 应用实例：青少年酒精使用轨迹

在 14、15、16 岁三个时间点测量青少年酒精使用频率，拟合随机斜率模型的结果：

- **固定效应**：总体截距 = 0.651（14 岁平均使用水平显著不为零）；时间斜率 = 0.271（平均每年增加 0.271 个单位）
- **随机效应**：截距方差 = 0.624（基线差异显著）；斜率方差 = 0.151（增长速度存在个体差异）
- **截距-斜率协方差** = -0.07（负相关：14 岁时起点越高，随后增长越慢）
- **Level 2 预测变量**：COA（酗酒者子女）显著预测更高的初始使用量（$\gamma_{01} = 0.743$），但对增长速度无显著影响

## 十二、潜增长模型（LGM）— SEM 视角

### 12.1 潜变量作为增长参数

在 LGM 中，数据为宽格式，每个时间点的测量为**显变量（Indicator）**，通过限制因子载荷识别两个潜变量：

- **潜截距因子（Latent Intercept）**：所有时间点载荷固定为 **1**，代表个体不随时间变化的本质水平
- **潜斜率因子（Latent Slope）**：载荷根据时间点固定，如四波等距测量设为 **0, 1, 2, 3**，代表线性变化轨迹

**lavaan 代码示例（四波线性增长）：**

```r
model <- '
  i =~ 1*y1 + 1*y2 + 1*y3 + 1*y4   # 截距因子，载荷全为1
  s =~ 0*y1 + 1*y2 + 2*y3 + 3*y4   # 斜率因子，载荷为时间分数
'
fit <- growth(model, data = widedat)
```

LGM 关心的核心参数：截距均值、斜率均值、截距方差、斜率方差、截距-斜率协方差，以及每个时间点的残差方差——与 MLM 估计的内容高度对应，只是使用潜变量语言表达。

### 12.2 纵向测量等价性（Longitudinal Measurement Invariance）

在比较潜变量随时间的变化之前，必须确认"这个构念在不同时间点是否以相同方式被测量"：

- **弱等价（Metric）**：跨时间点的因子载荷相等 → 保证潜在因子的单位一致
- **强等价（Scalar）**：跨时间点的载荷 + 截距均相等 → 是比较潜变量**均值变化**的必要前提
- 若强等价违背，观察到的"增长"可能只是指标本身漂移，而非潜在构念的真实变化

### 12.3 LGM 相比 MLM 的核心优势

- **测量误差的精细拆解**：将观测变异显式分解为"真实得分变异"和"测量误差变异"，估计的截距与斜率是在纠正随机测量误差后的结果，提高参数可靠性与效度
- **残差结构灵活**：允许残差项在不同时间点具有不同方差（异方差性），或相邻时间点残差存在自相关（捕捉练习效应、疲劳效应）
- **模型拟合可评估**：提供全局拟合指数（CFI、TLI、RMSEA、SRMR），可直接判断增长曲线形态是否契合观测数据

### 12.4 高级扩展

- **二阶潜增长模型（Second-order LGM）**：当每个时间点的指标本身是潜变量时（如由 5 个题目组成的抑郁量表），先建立测量模型，再在潜变量层面建立增长模型，同时检验测量不变性
- **增长混合模型（Growth Mixture Modeling, GMM）**：结合潜类别分析（LCA）与增长模型，假设总体由具有不同增长轨迹的子群组成（如"快速反应者"vs."慢性低反应者"），自动识别潜在发展轨迹分类

## 十三、MLM vs. SEM/LGM 决策指南

### 13.1 重要前提：简单增长模型下两者往往等价

对于简单线性增长模型，MLM 和 LGM 在很多情形下会得到数值上**相同或几乎相同**的结果。主要差异来源于残差结构的默认设定：

- **MLM**：通常默认 Level-1 残差方差**同质**（各时间点相同）
- **LGM/SEM**：各时间点残差方差默认**自由变化**（允许异方差）

因此两者在"简单线性增长"上常等价，但在更细的误差结构设定上可能出现差异。

### 13.2 决策清单

- **时间点数量多（≥10 次）或时间间隔不规则** → 优先 **MLM**（将时间视为连续预测变量，天然处理不平衡数据）
- **时间点较少（3–8 次）且步长规则** → 优先 **LGM/SEM**
- **关注多变量并行增长、中介效应或复杂因果路径** → 优先 **LGM/SEM**
- **研究对象是潜在构念（如抑郁、动机），每时间点有多个指标** → 必须用 **LGM/SEM**（显式校正测量误差，检验测量等价性）
- **小样本（N < 100）** → 优先 **MLM**（REML 估计更稳健；LGM 可能无法收敛或出现 Heywood Case）
- **大样本（N > 200）且存在潜在子群** → 优先 **GMM**
- **需要评估整体模型拟合** → **SEM**（CFI、RMSEA、SRMR）；**MLM** 只能用 AIC/BIC 相对比较，无法判断绝对拟合
- **更高层嵌套（时间点在学生中，学生在学校中）** → **MLM**（多层嵌套更自然）
- **时间分数本身不确定，需由数据估计** → **LGM**（斜率载荷可设为自由参数；MLM 中时间是固定输入值）

### 13.3 一句话判断

- 只需描述两波前后变化 → **change score / ANCOVA**
- 个体成长轨迹 + 数据有缺测/时间不齐/层级结构 → **MLM**
- 潜在构念的变化 + 认真处理测量误差与测量等价性 → **SEM/LGM**

> **前沿整合**：随着 Mplus 等软件的发展，**多层结构方程模型（MSEM）**已实现 MLM 与 SEM 的整合。最好的方法不是"统计上最高级"的那个，而是最贴合研究问题与数据生成机制的那个。

## 十四、时间度量（Time Metric）

时间编码不只是"wave = 0,1,2,3"的技术细节，它直接决定**截距的实质含义**。

- **Wave number**：简单直观，但截距只代表"第一波时的水平"，与个体真实年龄脱钩
- **Age**（实际年龄）：截距代表某一特定年龄的水平，更贴近生物/发展意义，但不同人的年龄范围可能不重叠
- **Days since treatment / time since event**：以事件为零点，截距代表干预/事件发生时的状态，适合临床研究
- **Years to death（向终点倒数）**：截距代表死亡前某时间点的状态，适合终末期疾病研究

**零点的影响：**

- LGM 中截距因子的载荷 = 0 的那个时间点就是"截距代表的那一刻"；零点移动不改变斜率，但会改变截距均值和截距-斜率协方差
- MLM 中同理：时间变量中心化的位置决定了 $\gamma_{00}$ 的解释

## 十五、Within-Person / Between-Person 分解

### 15.1 为什么必须分解

在纵向 MLM 中，一个时变协变量（Time-varying covariate, TVC）的系数往往是**组内效应（within-person effect）和组间效应（between-person effect）的混合体**。两者不拆开，系数可能误导。

**经典情景**：压力与负面情绪的关系

- **组间（Between-person）**：长期高压力的人，平均负面情绪也高
- **组内（Within-person）**：某人在压力高于自己均值的那天，负面情绪是否也更高？

这两个问题答案可能相同，也可能方向相反——混在一起的系数无法区分。

### 15.2 Person-Mean Centering（组内中心化）

做法：将每个人的时变 TVC 分解为：

$$X_{ti} = \bar{X}_i + (X_{ti} - \bar{X}_i)$$

- **$\bar{X}_i$（个体均值）**：进入 Level 2，捕捉**组间效应**（who 效应：高压力的人 vs. 低压力的人）
- **$X_{ti} - \bar{X}_i$（组内离差）**：进入 Level 1，捕捉**组内效应**（when 效应：此人在压力高于自己均值时）

**好处**：

- Level 1 的系数纯粹反映组内动态变化，不受个体间稳定差异污染
- 如果组内效应 ≠ 组间效应，强行用未中心化变量会导致参数偏倚

## 十六、Change Score 为何会误导？

### 16.1 测量层面：两个带误差的分数直接相减

设两次测量为 $Y_1 = T_1 + e_1$，$Y_2 = T_2 + e_2$，则：

$$\Delta = Y_2 - Y_1 = (T_2 - T_1) + (e_2 - e_1)$$

Change score 的"误差项"是 $e_2 - e_1$，若两次误差独立：

$$\text{Var}(e_2 - e_1) = \text{Var}(e_2) + \text{Var}(e_1)$$

**误差直接叠加**，change score 往往比任一单次测量都更不稳定。

此外，change score 与 baseline 存在机械负相关：

$$\text{Cov}(\Delta, Y_1) = \text{Cov}(Y_2 - Y_1, Y_1) = \text{Cov}(Y_2, Y_1) - \text{Var}(Y_1)$$

只要 $\text{Cov}(Y_2, Y_1) < \text{Var}(Y_1)$（即跨时相关 < 1），这个协方差通常为负 → baseline 高的人更容易显得"涨得少"，baseline 低的人更容易显得"涨得多"，这不一定反映真实机制。

### 16.2 因果层面：在观察性研究中常估计不到你想要的效应

- 在非随机化数据中，将 change score 作为结果变量回归某个基线暴露，通常不估计一个有明确因果含义的效应
- 当基线结局同时扮演 confounder 或 mediator 时，change score 系数甚至可能与真实 total/direct effect 方向相反
- **结论**：change score 不是永远不能用，而是在非随机化因果问题里，它常常不是你以为的 estimand

## 十七、Residualized Change 的具体含义与例子

Residualized change = 先用前测预测后测，再看实际后测与预测值的偏差：

$$\hat{Y}_{2i} = \hat{\beta}_0 + \hat{\beta}_1 Y_{1i}, \qquad R_i = Y_{2i} - \hat{Y}_{2i}$$

$R_i = 0$ **不代表"没有变化"**，而代表"此人的变化正好等于同样基线人群的平均预期变化"。

**具体例子**（全班回归结果：$\hat{Y}_{\text{post}} = 20 + 0.7 \times Y_{\text{pre}}$，斜率 < 1 反映向均值回归）：

- **学生 A**：pre = 40，post = 50；predicted = 48 → raw change = **+10**，residualized = **+2**
- **学生 B**：pre = 80，post = 90；predicted = 76 → raw change = **+10**，residualized = **+14**
- **学生 C**：pre = 70，post = 69；predicted = 69 → raw change = **−1**，residualized = **0**

A 和 B 的绝对进步相同（均为 +10），但 residualized change 告诉你：B 相对同等基线的人超出预期 14 分，而 A 只超出 2 分。C 虽然退步了 1 分，但这正好是同等基线人群的平均变化，所以 residualized change = 0（并不算"比预期更差"）。

> **一句话抓住本质**：change score = absolute change；residualized change = baseline-adjusted deviation from expected posttest。

## 十八、MLM 与 LSEM 如何在数学上"解决"相关性

> **核心思想**：两者都不是把相关性"消掉"，而是将相关性**参数化**，显式放进模型里。

### 18.1 MLM：用 Random Effects 生成组内协方差

线性增长 MLM 的完整形式：

$$y_{tj} = \gamma_{00} + \gamma_{10}x_{tj} + u_{0j} + u_{1j}x_{tj} + r_{tj}$$

对同一个体 $j$ 在时间点 $t$ 和 $s$ 的协方差（假设给定 random effects 后 Level-1 残差独立）：

$$\text{Cov}(y_{tj}, y_{sj}) = \tau_{00} + x_{tj}\tau_{01} + x_{sj}\tau_{01} + x_{tj}x_{sj}\tau_{11}$$

- **$\tau_{00}$**：共享随机截距带来的相关
- **$\tau_{11}$**：共享随机斜率带来的相关
- **$\tau_{01}$**：截距与斜率的耦合

**最简单的 random intercept only 情况：**

$$\text{Cov}(y_{tj}, y_{sj}) = \tau_{00} \quad (t \neq s)$$

所有观测共享同一个 $u_{0j}$，因此自然相关——与 cluster data 里的 block covariance 逻辑完全一致。

**密集追踪时还需加 serial correlation（AR(1)）：**

$$\text{Cov}(r_{tj}, r_{sj}) = \sigma^2 \rho^{|\text{time}_t - \text{time}_s|}$$

于是总协方差在 random effects 之上再叠加时间衰减的残差相关。

### 18.2 LSEM：用 Latent Factors 和残差协方差结构生成相关性

LSEM 的模型隐含协方差结构：

$$\Sigma(\theta) = \Lambda \Psi \Lambda' + \Theta$$

对时间点 $t$ 载荷向量 $\lambda_t = [1, x_t]$，任意两时间点的协方差为：

$$\text{Cov}(y_{tj}, y_{sj}) = \psi_{00} + x_t\psi_{01} + x_s\psi_{01} + x_tx_s\psi_{11} + \theta_{ts}$$

- **$\psi_{00}, \psi_{11}, \psi_{01}$**：latent intercept/slope 的方差和协方差（$\Psi$ 矩阵元素）
- **$\theta_{ts}$**：残差协方差（$\Theta$ 矩阵元素）——同题项跨时间的 residual dependence 可直接放在这里，或用 method factor 建模

### 18.3 MLM 与 LSEM 参数的一一对应

两个协方差公式结构完全相同，参数直接对应：

- $\text{Var}(u_0) = \tau_{00} \;\leftrightarrow\; \text{Var}(\eta_\text{intercept}) = \psi_{00}$
- $\text{Var}(u_1) = \tau_{11} \;\leftrightarrow\; \text{Var}(\eta_\text{slope}) = \psi_{11}$
- $\text{Cov}(u_0, u_1) = \tau_{01} \;\leftrightarrow\; \text{Cov}(\eta_\text{intercept}, \eta_\text{slope}) = \psi_{01}$
- $\text{Var}(r_t) = \sigma_t^2 \;\leftrightarrow\; \theta_{tt}$

这正是简单线性增长模型下两者数值结果常常相同的根本原因。差异主要来自扩展方向：**MLM** 更自然处理高层嵌套和不规则测量时间；**LSEM** 更擅长显式测量模型、测量等价性检验，以及把时间分数本身作为自由参数估计。

## 十九、潜在变化分数模型（Latent Change Score Model, LCS）

### 19.1 核心思想

LCS 模型将纵向过程显式分解为：

$$Y_t = Y_{t-1} + \Delta Y_t$$

通过引入**幻影变量（Phantom Variables）**，将每一时段的变化量 $\Delta Y_t$ 作为独立的潜变量隔离出来。它正好卡在 raw change、ANCOVA、ARCL 和 LCM 之间，是串联各类变化模型的桥梁。

### 19.2 两种变化成分

- **恒定变化（Constant Change）**：每个时间段增加固定量，类似 LCM 的线性斜率
- **比例变化（Proportional Change）**：当前变化量受前一状态影响（自反馈机制，系数 $\beta$），可模拟增长的加速或减缓：

$$\Delta Y_t = \alpha + \beta Y_{t-1}$$

若 $\beta < 0$（负反馈）→ 高水平时增长放缓，低水平时增长加速，系统趋向平衡；若 $\beta > 0$ → 发散增长

### 19.3 双变量 LCS（Bivariate LCS）

允许两个变量的变化量相互预测，建模**动态因果反馈**：

$$\Delta X_t = \alpha_X + \beta_X X_{t-1} + \gamma_{YX} Y_{t-1}$$
$$\Delta Y_t = \alpha_Y + \beta_Y Y_{t-1} + \gamma_{XY} X_{t-1}$$

- **$\gamma_{YX}$**：$Y$ 的前一水平对 $X$ 的变化的影响（交叉滞后效应）
- 可以同时检验两方向的动态影响，比传统 ARCL 更直接区分"水平驱动变化"的机制

---

## 二十、MLM ↔ SEM/LGM 统一图景与翻译词典

> **一句话核心**：MLM 是"条件表示"（给定每个人的随机截距/斜率后，观测如何生成）；SEM/LGM 是"边际/潜变量表示"（直接描述重复测量向量的均值和协方差由哪些潜变量生成）。对于线性高斯 growth model，两者是**同一个** $E(\mathbf{y}_i)$ 和 $\text{Var}(\mathbf{y}_i)$ 在不同坐标系下的写法。

### 20.1 从条件到边际：关键推导步骤

把 MLM 写成对"一个人整条轨迹"的矩阵形式：

$$\mathbf{y}_i = \mathbf{X}_i\beta + \mathbf{Z}_i\mathbf{b}_i + \mathbf{e}_i, \quad \mathbf{b}_i \sim N(\mathbf{0}, \mathbf{G}), \quad \mathbf{e}_i \sim N(\mathbf{0}, \mathbf{R}_i)$$

将随机效应 $\mathbf{b}_i$ 积分掉，得到观测向量的**边际分布**：

$$E(\mathbf{y}_i) = \mathbf{X}_i\beta \qquad \text{Var}(\mathbf{y}_i) = \mathbf{Z}_i\mathbf{G}\mathbf{Z}_i' + \mathbf{R}_i$$

SEM/LGM 的边际矩结构（$\mathbf{y}_i = \Lambda\eta_i + \varepsilon_i$，$\eta_i = \alpha + \zeta_i$）：

$$E(\mathbf{y}_i) = \Lambda\alpha \qquad \text{Var}(\mathbf{y}_i) = \Lambda\Psi\Lambda' + \Theta$$

两边形式完全相同，映射关系一目了然。

### 20.2 翻译词典

- **$\mathbf{Z}_i$（MLM random-effects design matrix）** $\;\leftrightarrow\;$ **$\Lambda$（SEM factor loading matrix）**
- **$\mathbf{b}_i$（random intercept/slope）** $\;\leftrightarrow\;$ **$\zeta_i$（latent growth factor 的个体偏差）**
- **$\mathbf{G}$（random-effects covariance matrix）** $\;\leftrightarrow\;$ **$\Psi$（latent factor covariance matrix）**
- **$\mathbf{R}_i$（Level-1 residual structure）** $\;\leftrightarrow\;$ **$\Theta$（indicator residual structure）**
- **$\mathbf{X}_i\beta$（fixed effects）** $\;\leftrightarrow\;$ **$\Lambda\alpha$（latent factor means）**

### 20.3 四波线性增长的具体展开

假设 4 个时间点 $t = 0, 1, 2, 3$，设计矩阵与载荷矩阵完全相同：

$$\mathbf{Z}_i = \Lambda = \begin{bmatrix} 1 & 0 \\ 1 & 1 \\ 1 & 2 \\ 1 & 3 \end{bmatrix}$$

- **MLM 读法**：$y_{it} = \gamma_{00} + \gamma_{10}t + u_{0i} + u_{1i}t + r_{it}$，其中 $[1, t]$ 是 random-effects design 的行向量
- **SEM 读法**：$\mathbf{y}_i = \Lambda\eta_i + \varepsilon_i$，其中 $\Lambda$ 的每一行正是 $[1, t]$

**只要令** $\alpha_0 = \gamma_{00}$，$\alpha_1 = \gamma_{10}$，$\zeta_{0i} = u_{0i}$，$\zeta_{1i} = u_{1i}$，两者完全等同。

> **最值得记住的一句话**：MLM 里的 $[1, t]$ 行向量，到了 SEM 里就变成了 $\Lambda$ 的一行。

### 20.4 只有随机截距时（与 null model 的对应）

对 3 人 cluster，MLM 协方差矩阵：

$$\mathbf{V}_j = \tau_{00}\mathbf{J}_3 + \sigma^2\mathbf{I}_3 = \begin{bmatrix} \tau_{00}+\sigma^2 & \tau_{00} & \tau_{00} \\ \tau_{00} & \tau_{00}+\sigma^2 & \tau_{00} \\ \tau_{00} & \tau_{00} & \tau_{00}+\sigma^2 \end{bmatrix}$$

等价的 SEM：一个 intercept-only latent factor，载荷向量全为 1：

$$\lambda = \begin{bmatrix}1\\1\\1\end{bmatrix}, \quad \text{Var}(\eta_j) = \tau_{00}, \quad \text{Var}(\varepsilon_j) = \sigma^2\mathbf{I}_3$$

$$\text{Var}(\mathbf{y}_j) = \lambda\tau_{00}\lambda' + \sigma^2\mathbf{I}_3 = \tau_{00}\mathbf{J}_3 + \sigma^2\mathbf{I}_3 \quad \checkmark$$

### 20.5 Level-2 预测变量在两套语言中的翻译

设 $w_i$ 为时间不变协变量（TIC）：

**MLM 写法（代入后的 reduced form）：**

$$y_{it} = \underbrace{\gamma_{00} + \gamma_{01}w_i}_{\text{main effect}} + \underbrace{\gamma_{10}t + \gamma_{11}(w_i \cdot t)}_{\text{cross-level interaction}} + u_{0i} + u_{1i}t + r_{it}$$

- **$w_i$ 预测随机截距** → 在 observed equation 中表现为 $w_i$ 的**主效应**
- **$w_i$ 预测随机斜率** → 在 observed equation 中表现为 $w_i \times t$ 的**跨层交互效应**

**SEM 写法（等价表达）：**

$$\eta_{0i} = \alpha_0 + \gamma_{01}w_i + \zeta_{0i}, \qquad \eta_{1i} = \alpha_1 + \gamma_{11}w_i + \zeta_{1i}$$

> MLM 说的"cross-level interaction"和 SEM 说的"predict latent slope factor"，是同一个效应。

### 20.6 两者何时不再等价

等价关系成立需要：线性高斯、时间编码一致、**残差结构设定一致**、**无额外测量模型**。

- **残差结构不一致**：LGM 默认各时间点残差方差自由变化（$\Theta$ 为对角但不等值）；MLM 默认 Level-1 残差同质（$\mathbf{R}_i = \sigma^2\mathbf{I}$）。两者一旦设定不同，就不是同一个模型
- **SEM 有显式测量模型**：每个时间点若是多指标的潜构念（如 5 道题测抑郁），SEM 可再往前加一层测量方程；MLM 若仍用总分建模，就不再同构

> **更精确的表述**：MLM 与 LGM 的等价，是在**观测层面的线性 growth model** 上成立；一旦 SEM 进一步显式处理测量误差和测量模型，就超出了简单一一翻译的范围。
